# Calculadora Android - Programación III

Proyecto de calculadora básica en Kotlin.

## Ejecutar el proyecto

1. Descomprimir el archivo .zip
2. Abrir la carpeta en Android Studio
3. Esperar a que sincronice Gradle
4. Ejecutar la app (Shift + F10)

## Características

- Operaciones: +, -, ×, ÷
- Tema oscuro
- Min SDK 21

## Requisitos

- Android Studio
- SDK de Android
